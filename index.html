Free, open-source ruler app for mac with web-development in mind.

Choose your unit

Working in CSS often means using EM for components. It is possible to switch between both PX and EM, and to change the base font value.

Works with devtools open

Linear isn't a browser extension, so it doesn't add a CSS overlay to your page. It's super easy to have rulers open while tweaking values in the inspector.

Duplicate rulers

Get the proportions right once, then copy the rulers to make sure your margins are correct.

Show and hide rulers

Toggle rulers using a global shortcut.

Dark and light

Working on a white background? Juste change the theme of the ruler.

Multiple rulers

Create as many rulers to make sure all your elements are aligned correctly.
